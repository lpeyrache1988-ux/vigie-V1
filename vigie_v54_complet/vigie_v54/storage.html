Sauvegarder vos données

Vos dossiers et préférences sont stockés sur votre appareil (localStorage). Vous pouvez exporter un fichier de sauvegarde et le réimporter plus tard.
Conseils ▾
  • Exportez avant chaque mise à jour importante.
  • En import, le mode fusion n’écrase pas ce qui existe déjà.
  • Le mode debug affiche les tags techniques (Flow/Score) pour les tests.
